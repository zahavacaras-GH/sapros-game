# SAPROS — Image Prompts (70 images)

> **How to use this file:** Copy the **MASTER STYLE BLOCK** + the relevant **CHARACTER REFERENCE SHEETS** + the **per-node prompt** into ChatGPT (or whichever image model you're using). Paste them as one continuous prompt. Output to 4:3 aspect, save as `<filename>.webp`, place in the project's `/images/` folder.
>
> **Critical for consistency:** Every image with a recurring character must include that character's reference sheet *verbatim*. Don't paraphrase — paraphrasing is what makes faces drift between generations.
