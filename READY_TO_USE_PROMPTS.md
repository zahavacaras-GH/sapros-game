# SAPROS — Ready-to-Use Image Prompts

> **How to use this file:**
> 1. Find the image you want to generate (organized by act).
> 2. Copy the entire prompt block (everything inside the gray code box).
> 3. Paste it into ChatGPT (or your image model).
> 4. When the image is generated, save it with the **exact filename** listed above the prompt (e.g., `t1.1_arrival.webp`).
> 5. Drop all images into a single folder called `images/` when you're done.
>
> **Each prompt is self-contained** — style, character descriptions, and scene are all baked in. No assembly needed.
>
> **For 4:3 aspect:** if the image model lets you choose, pick 4:3. If not, generate at default and we can crop later.
